AUTOSAR Mode Management and State Transition Testing
This project focuses on understanding AUTOSAR Mode Management and validating correct ECU mode transitions such as OFF, ON, and transitional states. The objective is to design structured test cases, identify failure scenarios, and automate state transition testing to ensure reliable and safe vehicle ECU behavior. The project provides practical exposure to ECU state machines and mode-based validation techniques used in automotive systems.
