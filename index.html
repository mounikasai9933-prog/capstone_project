Project Summary:

This project focuses on validating Automotive ECU Mode Management and State Transitions using automated testing techniques. The system ensures proper switching between ECU modes such as OFF, INIT, RUN, and SLEEP. Automated test scripts were developed using Appium and Java to verify correct state transitions, diagnostic session handling, and error management to prevent system malfunction in automotive environments.

Objective:

To ensure reliable ECU behavior by validating correct mode transitions and detecting invalid state changes using automation testing.
